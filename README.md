## What does a _ui-automation-bootstrap_ test look like?
```java
@Test
@Mock(clazz = MockExampleModel.class)
public void shouldPerformSearchSuccessfully() {
    open("http://google.com");

    googleSearchPage
            .searchComponent()
            .searchFor("mock");

    googleResultsPage
            .searchComponent()
            .self()
            .shouldHave(value("mock"));
}
```

## About
